👋 Welcome to the User Management API

This lightweight ASP.NET Core Web API lets you manage user records using simple CRUD operations. It uses an in-memory data store, so no database setup is required.

🚀 Quick Start

To test the API, you can use:

  • Swagger UI — interactive documentation
  • Postman — recommended for reviewers

📘 Swagger Documentation

Open Swagger UI at:

/swagger

🧪 Postman Testing

Import the included UserManagementAPI.postman_collection.json file.
Then set the baseUrl variable to your local port, for example:

https://localhost:5001

🔧 Available Endpoints

GET /users — Get all users
GET /users/{id} — Get a user by ID
POST /users — Create a new user
PUT /users/{id} — Update a user
DELETE /users/{id} — Delete a user
